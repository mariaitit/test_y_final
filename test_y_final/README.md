# Автоматизация теста к API — создание и получение заказа

В проекте реализован автотест для проверки сценария:
клиент создаёт заказ и по треку заказа получает данные о нём.

# Шаги теста
1. Выполняется POST-запрос на создание заказа.
2. Из ответа сохраняется номер трека заказа.
3. Выполняется GET-запрос на получение заказа по треку.
4. Проверяется, что код ответа равен 200.

# Структура проекта
- configuration.py — базовый URL API
- data.py — данные для создания заказа
- sender_stand_request.py — функции для отправки запросов
- tests/test_create_and_get_order.py — автотест
- requirements.txt — зависимости проекта
- .gitignore — файлы и папки, исключённые из репозитория

# Подготовка и запуск
1. Установите Python 3
2. Установите зависимости
3. Запустите тесты
